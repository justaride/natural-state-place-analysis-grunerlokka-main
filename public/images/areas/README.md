# Area Hero Images

This directory contains hero images for the analysis overview cards.

## Required Images:

### 1. grunerlokka.jpg
- **Usage**: 2024 Årsrapport card + header in analysis page
- **Dimensions**: Recommended 1920x1080 or similar 16:9 ratio
- **Description**: Representative image of Grünerløkka area

### 2. sammenligning-collage.jpg
- **Usage**: Områdesammenligning 2024 card + header
- **Dimensions**: Recommended 1920x1080 or similar 16:9 ratio
- **Description**: Collage of 4 images showing:
  - Grünerløkka
  - Bjørvika
  - Sentrum
  - Majorstuen

## Individual Area Images (for collage creation):
- bjørvika.jpg
- sentrum.jpg
- majorstuen.jpg

## Placement Instructions:
1. Place images in this directory
2. Images will be automatically used in:
   - `/analyser` overview page cards
   - Individual analysis page headers
3. Images use Next.js Image optimization

## Format Guidelines:
- Format: JPG or PNG
- Max file size: 2MB recommended
- Aspect ratio: 16:9 preferred
- Quality: High resolution for retina displays
